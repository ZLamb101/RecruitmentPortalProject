Vesta Recruitment Portal Prototype

Made by: Zane Lamb, Benjamin Upton and Jordan Drumm

In order to run and test this web portal, you must first have docker installed. Once installed and runnning, navigate to the directory where this readme is located via the command prompt or similiar and type docker-compose up. Once all the dependencies have finished downloading and installing, you can navigate to localhost:8000 via a web browser where you will find the login page for the site. From here, you are free to create an account, login, and test out the portal. Some dummy accounts are set up as follows:
